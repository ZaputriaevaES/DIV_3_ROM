# DIVISIBILITY_BY_3 & ROM
 Two simple projects for the implementation of divisibility testing at 3 and memory module.

###  How to build DIVISIBILITY_BY_3
in folder div_by_3/
```
    make div_by_3
    vvp div_by_3
```
###  How to run GTKWave
in folder div_by_3/
```
   gtkwave dump.vcd
```

###  How to build ROM
in folder memory/
```
    make memory
    vvp memory
```
###  How to run GTKWave
in folder div_by_3/
```
   gtkwave dump.vcd
```
